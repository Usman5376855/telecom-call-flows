## Overview

Welcome!

This repository contains a growing collection of telecommunications call flows, signaling procedures, architecture diagrams, and technical notes covering LTE, EPC, IMS, VoLTE, VoNR, and 5G Core Networks.

The objective is to provide engineers, students, and telecom professionals with practical learning material and troubleshooting references.
